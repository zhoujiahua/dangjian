    刘家义主持召开调研座谈会 听取关于做好我省经济工作的意见建议
    发布时间：2019-06-10 15:33:30　点击数量:201

　　7月21日至22日，省委书记刘家义主持召开调研座谈会，分别听取企业、金融和省直有关部门汇报，研究分析当前全省经济社会发展形势，听取关于做好下一步经济工作的意见建议，强调要坚持以习近平新时代中国特色社会主义思想为指导，深入贯彻习近平总书记视察山东系列重要讲话和重要指示精神，坚定践行新发展理念，深化改革开放，加快制度创新，抢抓发展机遇，顺势而为、逆势而上，推动我省经济高质量发展。

　　座谈会上，20余名来自我省部分企业、金融领域、省直部门（单位）的主要负责同志，就我省经济运行中存在的问题进行了分析研判，围绕实体经济转型升级、防范化解重大风险、推进制度创新和体制机制改革、加快新旧动能转换等方面提出对策。大家紧密结合我省实际，直奔主题、剖析问题，各抒己见、有的放矢，提出务实管用的意见建议。刘家义认真听取大家发言并详细记录，不时询问有关问题，同与会同志深入讨论，共同研究助推我省经济高质量发展的政策措施，对大家提出的意见建议给予充分肯定，表示要认真梳理研究并予以吸收，体现到下一步经济工作中。

　　刘家义说，今年以来，通过各方面努力，我省经济运行保持在合理区间。面对当前经济发展新情况新问题，我们要坚持稳中求进工作总基调，在稳中调，在调中进。坚定不移落实新发展理念，坚定不移推动高质量发展，坚定不移加快新旧动能转换，坚定不移推动乡村振兴，坚定不移加快海洋经济发展，坚定不移加快军民融合，坚定不移打好三大攻坚战。要进一步深化改革扩大开放，加快进度、加大力度推动制度创新，增加有效制度供给，以制度创新释放改革发展动力活力。要进一步防范化解重大风险，加强地方金融改革，强化市场监管，增强金融支持实体经济发展、服务新旧动能转换的能力。企业要认清形势、抢抓机遇，深挖潜力、积极进取，做到顺势而为、逆势而上。党员干部特别是领导干部要深入农村、深入企业，察真情、听民声，帮助基层和群众解决实际问题，在实践中增强本领。各级各有关部门要持续深化“放管服”改革，努力创造一流营商环境，积极构建亲清新型政商关系，培养造就一大批企业家，助推我省企业做优做强做大，凝聚各方力量推动我省经济高质量发展。

省领导杨东奇、王清宪、王书坚，省直有关部门负责同志参加座谈会。
